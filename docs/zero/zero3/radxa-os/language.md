# 语言设置

1. 安装简体中文字体：

```
sudo apt-get install fonts-wqy-zenhei
```

2. 设置系统语言为中文桌面：

```
sudo dpkg-reconfigure locales
```

3. 选择中文语言

![set language1 ](/img/zero/zero3/set-language1.webp)

![set language2 ](/img/zero/zero3/set-language2.webp)

4. 重启系统后即可

```
sudo reboot
```

其他语言的配置可以参照上述配置步骤
