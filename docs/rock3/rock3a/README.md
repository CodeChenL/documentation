# ROCK 3A

## 产品介绍

- ROCK 3A 配备了四核 Cortex-A55 ARM 处理器和 LPDDR4 运行内存， 最高支持 4K@60 的 HDMI、MIPI DSI、MIPI CSI、带麦克风的 3.5 毫米耳机插孔、USB 端口、千兆以太网、PCIe 3.0、PCIe 2.0、40 pin GPIO 扩展座、RTC。 此外，ROCK 3A 还支持 PD 和 QC 协议供电。
