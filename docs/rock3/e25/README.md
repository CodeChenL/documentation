# E25

## 介绍

Radxa E25 是 Radxa CM3I 的功能拓展板，拥有丰富双 2.5G 以太网、mini-PCIe、B-Key 等丰富接口。

- Radxa E25
  ![Radxa E25](/img/cm3i/e25/e25-overview.webp)

## 外设概述

- 1x USB3.0 Host/OTG
- 1x B Key
- 1x PCI-E MINI
- 1x TF Card Slot
- 2x 2.5 Gigabit Ethernet
- 1x 26 PIN GPIO
- RGB LED Light
- 1x Power Input
