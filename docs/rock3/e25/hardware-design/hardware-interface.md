## 26 PIN GPIO

#### GPIO 电压

| GPIO        | 电压 | 最高  |
| ----------- | ---- | ----- |
| 所有的 GPIO | 3.3V | 3.63V |
| SARADC_IN5  | 3.3V | 3.3V  |
