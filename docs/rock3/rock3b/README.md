# ROCK 3B

瑞莎 ROCK 3B 是一款基于 Pico-ITX 外形规格（100mm x 75mm）的单板计算机 (SBC)，与同类产品相比，不仅在性能方面更胜一筹，还更具出色的机械兼容性。
并且，ROCK 3B 不仅具有社区版本（RK3568），还提供专门定制的工业版本（RK3568J），其丰富的接口以及宽温度使用范围可为工业领域的客户提供可靠而强大的平台。
