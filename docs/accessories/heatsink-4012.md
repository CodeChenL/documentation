# Heatsink 4012 for ROCK 5B

Radxa Heatsink 4012是一款用于ROCK 5B的散热器，长40mm，宽40mm，高12mm。
![Heatsink 4012](/img/accessories/heatsink-4012-1.webp)

## 物理规格

- 孔径：3mm
- 尺寸：4 x 4 x 1.2 厘米
- 速度：4300rpm ± 10%
- 孔距：5.5 厘米
- 声音：18dB
- 材质：铝
- 轴承：合金轴承
- 电压：5V
