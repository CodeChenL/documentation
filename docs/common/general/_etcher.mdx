Balena Etcher 是一个跨平台且，用户界面友好的镜像文件烧写工具，我们推荐你使用它。
如果你使用 Windows，并且对 [Win32DiskImager](https://win32diskimager.org) 或 [Rufus](https://github.com/pbatard/rufus) 更熟悉，你也可以使用它们。

1. 下载 [Etcher](https://etcher.balena.io/#download-etcher/) 并安装。

   <Image
     src="/img/common/etcher/#model#-step1.webp"
     model={props.model}
     fallback="rock5a"
   />

2. 打开 Etcher，单击 `Flash from file` 以选择需要写入的镜像。

   <Image
     src="/img/common/etcher/#model#-step2.webp"
     model={props.model}
     fallback="rock5a"
   />

3. 点击 `Select target` 以选择设备，请注意小心选择。

   <Image
     src="/img/common/etcher/#model#-step3.webp"
     model={props.model}
     fallback="rock5a"
   />

4. 点击 `Flash!` 开始写入，然后等待写入进度条完成。

   <Image
     src="/img/common/etcher/#model#-step4.webp"
     model={props.model}
     fallback="rock5a"
   />

5. 当写入镜像成功时，Etcher 将会显示 `Flash Complete!`。
   <Image
     src="/img/common/etcher/#model#-step5.webp"
     model={props.model}
     fallback="rock5a"
   />

**如果刷写操作系统镜像错误, 请手动再试一次。**
