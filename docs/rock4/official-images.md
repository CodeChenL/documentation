# ROCK 4 系列官方镜像

ROCK 4 官方操作系统镜像可从以下方式下载：  

- **[ROCK 4 镜像发布网址](https://github.com/radxa/debos-radxa/releases/latest)** - 最新系统镜像。

关于用户名和密码：  
以下内容同样适用于 Ubuntu 服务端。
Lubuntu 默认有两个用户：root 和 rock，用户名和密码如下：

```
user password
rock    rock
root    root
```

如果镜像是自己创建的，默认用户为 linaro：

```
user password
linaro linaro
```
