## WIFI 使用

1. 首先进入ROOT用户模式。

```
sudo su
```

2. 打开WIFI

```
nmcli r wifi on
```

3. 扫描WIFI

```
nmcli dev wifi
```

4. 连接wifi网络

```
nmcli dev wifi connect "wifi_name" password "wifi_password"
```
